taggedPlus by Alex de Moure


A safari extension for tagged.com

  • Make "Navigation Bar" static, by making the Tagged menu bar fixed at top, also hides the ads.


  • Use "Browse Bar" as a dock, if you use "Browse" to search profiles, this option moves the browser at bottom-right, it's also auto hidding like a dock.


  • Use arrows as navigation keys, browse photos using the arrow keys: left arrow and right arrow.
  • Skins: There are times when the background color and text doesn't contrast, the extension offers an option to change "Skins" globally, the choices are: default skin, plus 4 skin designs, and of course the option to not replace it.
  • It also fixes the "Change Skin" module in Safari.


  • Move "Footer" to bottom in Home, by moving the page footer to the bottom in Home page, you'll free some space.
